# 📌 AdvancedTesla
A plug-in for the SCP:SL game that activates tessellations only for selected roles.

# 🛠️ Requirements
EXILED >= 9.5.1

# ⚙️ Configuration
```
tesla_advanced:
# Włącza lub wyłącza plugin.
  is_enabled: true
  # Debug Mode.
  debug: false
  # Lista teamow, które POWINNY aktywować bramki Tesli.
  activating_teams:
  - ChaosInsurgency
  - ClassD
  - SCPs

```
